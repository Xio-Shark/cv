## 💡 核心发现

### 1. 直方图均衡化的本质

**发现:**
- 直方图均衡化本质是CDF的线性化
- 自适应方法能避免全局方法的过度增强
- 对比度限制是关键参数

**公式理解:**
```
s = T(r) = (L-1) ∫[0,r] p(w)dw
```
其中CDF的导数就是概率密度函数PDF

### 2. DCT能量集中特性

**发现:**
- 图像能量高度集中在低频部分
- 实验2-3证明:只用0.38%系数可达30.95 dB
- DC分量代表平均亮度,是最重要的系数

**数据证明:**
```
实验2-3统计:
  前10个系数: 占总能量的X%
  前20个系数: 占总能量的Y%
  前1000个系数: 足以重建可识别图像
```

### 3. 分块DCT的实用价值

**发现:**
- 8×8分块是计算效率和压缩性能的最佳折中
- JPEG标准选择8×8有充分的理论和实践支持
- 块效应可通过后处理滤波缓解

**对比:**
```
4×4: 块效应小但压缩比低
8×8: 最佳平衡 ✓
16×16: 块效应严重且计算量大
```

```python
# 自己实现直方图计算
hist = np.zeros(256)
for pixel_value in image:
    hist[pixel_value] += 1

# 自己实现CDF
cdf = np.cumsum(hist)

# 自己实现映射
transform_map[i] = ((cdf[i] - cdf_min) / (total - cdf_min)) * 255
```

**DCT变换:**
```python
# 自己构建DCT矩阵
def create_dct_matrix(n):
    for i in range(n):
        for j in range(n):
            alpha_i = np.sqrt(1/n) if i==0 else np.sqrt(2/n)
            C[i,j] = alpha_i * np.cos((2*j+1)*i*π/(2*n))
    return C

# 自己实现2D DCT
F = C @ image @ C.T
```
